# Portfolio - Benjamin Siret

Portfolio minimaliste développé en HTML, CSS et JavaScript vanilla, sans frameworks ni bibliothèques externes. Cette approche reflète ma conviction qu'une compréhension approfondie des fondamentaux est essentielle.

## Caractéristiques

- Design responsive
- Performance optimisée
- Accessibilité soignée
- Code vanilla uniquement

## Technologies

- HTML5 sémantique
- CSS3 (Flexbox & Grid)
- JavaScript ES6+
- Aucune dépendance externe

## Points techniques notables

- Lazy loading des images
- Animations CSS natives
- Intersection Observer pour les effets de scroll
- SEO optimisé
