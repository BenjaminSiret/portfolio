Expérience et parcours

Mon parcours combine une formation technique avec une expérience significative de 2 ans chez Remixt (remixt.co), où j'ai notamment : - Développé des solutions multilingues optimisées - Conçu une API RESTful sécurisée - Créé des visualisations de données interactives - Participé à la modernisation d'une application legacy.Cette expérience m'a permis de développer une expertise particulière dans la création d'interfaces utilisateur performantes et la gestion de données complexes.

Philosophie

Mon approche du développement web repose sur trois piliers : - Une maîtrise approfondie des fondamentaux (JavaScript, HTML, CSS) - Une expertise dans les outils modernes quand ils apportent une réelle valeur - Un souci constant de la qualité (tests, performance, accessibilité). Cette approche me permet de créer des applications robustes et maintenables, adaptées aux besoins des utilisateurs comme des équipes de développement.

Projets

Quelques projets réalisés pendant ma formation. Ainsi que la PWA Equilibre, une application de suivi de santé mentale.

Contact

Intéressé par des projets qui mettent l'accent sur : - L'innovation technique responsable - L'expérience utilisateur - La qualité du code - Le travail d'équipe. N'hésitez pas à me contacter pour échanger sur vos projets ou défis techniques.

À propos de ce portfolio

Ce portfolio est construit uniquement avec HTML, CSS et JavaScript vanilla, sans framework ni librairie.
Cela reflète ma conviction qu'une compréhension approfondie des fondamentaux est essentielle pour tout développeur web.
